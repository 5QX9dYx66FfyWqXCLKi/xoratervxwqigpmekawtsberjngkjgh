Karir AGM Pulsa

Kami adalah perusahaan yang bergerak di bidang PULSA dan PPOB, yang sedang berkembang pesat, yang berlokasi di Indramayu. Saat ini mencari para profesional yang menyukai tantangan, pekerja keras dan sangat termotivasi untuk mengisi posisi berikut :
Junior Programmer (Jr Progr) :
  1. Pria / Wanita Max 27 thn
  2. Pendidikan minimum D3 Teknik Informatika / Sistem Informasi
  3. Menguasai Query Database (MySQL)
  4. Menguasai bahasa program Java / PHP (code igniter)
  5. Pengalaman membuat aplikasi Java / PHP (harap melampirkan contoh aplikasi yang pernah dibuat)
  6. Dapat bekerja baik dalam tim maupun sendiri
  7. Mempunyai integritas dan ketelitian yang tinggi dalam bekerja
  8. Mempunyai loyalitas yang tinggi terhadap perusahaan
  9. Biasa bekerja dengan target dan dalam tekanan
  10. Sanggup bekerja di atas jam kerja normal
Deskripsi Pekerjaan :
  1. Mengembangkan Aplikasi Java Desktop
  2. Mengembangkan Aplikasi Java Mobile
  3. Mengembangkan website menggunakan framework code igniter

Business Development Officer (BDO)
  1. Wanita
  2. Pendidikan D3 atau S 1 (semua jurusan, lebih diutamakan dari Bisnis Administrasi)
  3. Single, penampilan menarik
  4. Usia max. 25 thn
  5. Lebih disukai memiliki SIM A
  6. Memiliki pengalaman kerja di bidang yang sama min.1 thn, diutamakan di Perusahaan Telekomunikasi atau Finansial
  7. Mahir mengoperasikan Aplikasi Microsoft Office dan Internet
  8. Mempunyai kemampuan komunikasi yang baik dan berorientasi pada detil.
  9. Mempunyai integritas dan ketelitian yang tinggi dalam bekerja
  10. Dapat bekerja baik dalam Tim maupun sendiri
  11. Biasa bekerja dengan target, mampu bekerja dalam tekanan dan menangani beberapa pekerjaan sekaligus
  12. Mampu membuat proposal dan melakukan presentasi di hadapan calon mitra bisnis

Mobile Developer (Android dan iOS)
  1. Pria / Wanita Max 27 thn
  2. Pendidikan minimum D3 Teknik Informatika / Sistem Informasi
  3. Pengalaman min. 1 tahun di Startup & E-Commerce
  4. Berpengalaman kuat dan extensive di Android dan iOS, ObjectiveC dan Swift
  5. Berpengalaman mengimplementasi API logistic, payment gateway dan positioning /geolocation
  6. Memiliki pengetahuan mendalam tentang UIKit, Foundation, Core Data, Core Graphics,framework
  7. Berpengalaman mengimplementasi AppsFlyer atau produk analytic serupa
  8. Berpengalaman membangun aplikasi E-Commerce Android dan iOS devices dari nol sampai live di App Store
  9. Passionate dan menyukai dunia startup dan E-Commerce
  10. Dapat bekerja baik dalam tim maupun sendiri
  11. Mempunyai integritas dan ketelitian yang tinggi dalam bekerja
  12. Mempunyai loyalitas yang tinggi terhadap perusahaan
  13. Biasa bekerja dengan target dan dalam tekanan
  14. Sanggup bekerja di atas jam kerja normal

Staf Rekonsiliasi Data (Kode : Rekon)
  1. Pria / Wanita Max 27 thn
  2. Pendidikan minimum D3 Teknik Informatika / Sistem Informasi
  3. Menguasai Query Database (MySQL)
  4. Menguasai troubleshooting Database My SQL (Instalasi, setting, tune up)
  5. Menguasai MS Office Khususnya Ms Excel
  6. Dapat bekerja baik dalam tim maupun sendiri
  7. Jujur,teliti, mempunyai integritas dan loyalitas tinggi terhadap perusahaan
  8. Biasa bekerja dengan target dan dalam tekanan
