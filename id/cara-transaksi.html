FORMAT TRANSAKSI

Semua Transaksi dikirim ke Jabber / Telegram / Gtalk / Hangout / SMS Center / Nimbuzz Center. Adapun Format Transaksi yang disediakan adalah sebagai berikut :   

Format bebas bisa menggunakan karakter : 
[ . ] / [ , ] / [ # ] / [ * ] / [ sepasi ] / [ Campur ]
Contoh 
Titik : 10.081234567893.1234
Koma : 10,081234567893,1234
Pagar : 10#081234567893#1234
Bintang :10*081234567893*1234
Sepasi : 10 081234567893 1234
Campur : 10#08123456783*1234

  • Daftar Menjadi MASTER
    Format : <REG>#<AGM>#<NAMA/KONTER>#<KOTA>
    Contoh : DAFTAR#JOKO#JAKARTA

  • Transaksi Isi Pulsa
    Format : <koode>.<no hp>.<pin>
    Contoh : SS10.081234567890.1234

  • Transaksi Isi Pulsa Tanpa Kode
    Format : <nominal>.<no hp>.<pin>
    Contoh : 10.081234567890.1234

  • Transaksi Token PLN TANPA NO TUJUAN
    Format : <nominal>.<no token>.<no hp tujuan>.<pin>
    Contoh : PLN20.34012312345.1234

  • Transaksi Token PLN
    Format : <nominal>.<no token>.<no hp tujuan>.<pin>
    Contoh : PLN20.34012312345.081234567890.1234

  • Transaksi Isi Pulsa Ke No Hp dan Nominal Yang Sama Ke 2,3,4 dst
    Format : <koode>.<2,3,4,dst><no hp>.<pin>
    Contoh : SS10.2.081234567890.1234  
    Contoh Tanpa kode: 10.3.081234567890.1234

  • Cek Saldo
    Format : S.<pin>
    Contoh : S.1234

  • Cek Saldo Downline
    Format : Saldo.<id_agen>.<pin>
    Contoh : Saldo.AB007.<pin>

  • Cek Harga
    Format : H.KodeProduk.Pin
    Contoh : H.S.1234

  • Ganti Pin
    Format : GPIN.<pinbaru>.<pinlama>
    Contoh : PIN.SaMurai7094.1234
    Catatan : PIN Bebas boleh angkan / huruf disarankan kominasi angkahuruf dan panjang agar aman

  • Tambah Hp
    Format : HP.<nomorhpbaru>.<pin>
    Contoh : HP.0877412345.1234

  • Daftar Downline
    Format : D.<nama>.<nohpdownline>.<up>.<alamat>.<pin>
    Contoh : D.Kios Cell.0812222122.50.Bandung.1234

  • Kirim Saldo
    Format : KIRIM.<id_member>.<jumlah>.<pin>
    Contoh : Kirim.OK001.200000.5555

  • Cek Bonus/Rebate
    Format : BONUS.<pin>
    Contoh : BONUS.1234

  • Cek Status Transaksi
    Format : STATUS.<tujuan>.<pin>
    Contoh : STATUS.0812345789.1234

  • Komplain
    Format : INFO.<isipesan>.<pin>
    Contoh : INFO.isi 10rb ke 0812222121 blm mask tgl.18 tolong bntu.1234

  • Tiket Deposit
    Format : TIKET.<Jumlah>.<pin>
    Contoh : TIKET.500000.1234

  • Hapus Tiket
    Format : HTIKET.<pin>
    Contoh : HTIKET.1234

  • Blokir Downline
    Format : Blokir.<id_agen>.<pin>
    Contoh : Blokir.BJ003.<pin>

  • Buka Blokir 
    Format : Buka.<id_agen>.<pin>
    Contoh : Buka.BJ003.1234

  • Ubah Up/Mark up Harga
    Format : UBAH.<id_downline>.<upbaru>.<pin>
    Contoh : UBAH.IN002.100.1234

  • Ubah Up/Mark up Perkodeproduk
    Format : SETUP.<id_agen>.<kode>.<up>.<pin>
    Contoh : SETUP.TE001.S5.50.1234

  • Ubah Up/Mark up Pernominal
    Format : SETUP.<id_agen>.<nominal>.<up>.<pin>
    Contoh : SETUP.TE001.100000.300.1234

  • Ubah Up/Mark up Perkodeproduk SemuaAgen 
    Format : SETUP.ALL.<kode>.<up>.<pin>
    Contoh : SETUP.ALL.S5.50.1234

  • Ubah Up/Mark up Pernominal SemuaAgen
    Format : SETUP.ALL.<nominal>.<up>.<pin>
    Contoh : SETUP.ALL.100000.50.1234

  • Cek Tagihan PPOB
    Format : CEK.PLN/TELKOM/BPJS/HALO/PDAM/SPEEDY.<id_pelanggan>.<pin>
    Contoh : CEK.PLN.53531004567699.1234

  • Bayar Tagihan PPOB
    Format : BAYAR.<id_pelanggan>.<pin>
    Contoh : BAYAR.535210045676.1234

  • Tarik Saldo
    Format : TARIK.<id_member>.<jumlah>.<sebabtariksaldo>.<pin>
    Contoh : TARIK.IK008.50000.Salah Kirim.1234

  • Set Hp Utama
    Format : HPUTAMA.<nomorhpygterdaftar>.<pin>
    Contoh : HPUTAMA.0812345454511.<pin>

  • Hapus Nomor Paralel
    Format : HAPUSHP.<nomorparalel>.<pin>
    Contoh : HAPUSHP.0877788882222.<pin>

  • Cek Semua Nomor
    Format : SEMUAHP.<pin>
    Contoh : SEMUAHP.1234

  • Set Trx Tanpa Pin
    Format : SETPIN.<0 atau 1>.<pin>
    Contoh : SETPIN.1.1234

  • Ganti Nama Member
    Format : Gantinama.<namabaru>.<pin>
    Contoh : Gantinama.Abadi Cell.1234

  • Ambil Bonus REBATE
    Format : AMBIL.<pin>
    Contoh : AMBIL.1234

  • ADD Google Talk
    Format : GTALK.<id/account>.<pin>
    Contoh : GTALK.Joko@gmail.com.1234

  • ADD JABBER
    Format : JABBER.<id/account>.<pin>
    Contoh : JABBER.joko@jabbim.com.<pin>

  • ADD NIMBUZZ
    Format : NIMBUZZ.<id/account>.<pin>
    Contoh : NIMBUZZ.Raisa@nimbuzz.com.<pin>

  • ADD TELEGRAMBOT
    Format : TELEGRAM.<ID telegram>.<pin>
    Contoh : TELEGRAM.11251244412.1234
