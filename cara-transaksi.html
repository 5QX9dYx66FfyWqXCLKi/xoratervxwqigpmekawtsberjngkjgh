FORMAT TRANSAKSI

Semua Transaksi dikirim ke Jabber / Telegram / Gtalk / Hangout / SMS Center / Nimbuzz Center. Adapun Format Transaksi yang disediakan adalah sebagai berikut :

Format bebas bisa menggunakan karakter :
[ . ] / [ , ] / [ # ] / [ * ] / [ sepasi ] / [ Campur ]
Contoh
Titik : 10.081234567893.1234
Koma :10,081234567893,1234
Pagar : 10#081234567893#1234
Bintang :10*081234567893*1234
Sepasi : 10 081234567893 1234
Campur : 10#08123456783*1234

  • Daftar Menjadi MASTER
    Format : DAFTAR#NAMA/KONTER#KOTA
    Contoh : DAFTAR#JOKO#JAKARTA

  • Transaksi Isi Pulsa
    Format : koode.no hp.pin
    Contoh : SS10.081234567890.1234

  • Transaksi Token PLN TANPA NO TUJUAN
    Format : nominal.no token.no hp tujuan.pin
    Contoh : PLN20.34012312345.1234

  • Transaksi Token PLN
    Format : nominal.no token.no hp tujuan.pin
    Contoh : PLN20.34012312345.081234567890.1234

  • Transaksi Isi Pulsa Ke No Hp dan Nominal Yang Sama Ke 2,3,4 dst
    Format : koode.2,3,4,dstno hp.pin
    Contoh : SS10.2.081234567890.1234
    Contoh Tanpa kode: 10.3.081234567890.1234

  • Cek Saldo
    Format : SAL.pin
    Contoh : SAL.1234

  • CekSaldo Downline
    Format : Saldo.id_agen.pin
    Contoh : Saldo.AB007.pin

  • Cek Harga
    Format : CH.KodeProduk.Pin
    Contoh : CH.S.1234

  • Ganti Pin
    Format : GPIN.pinbaru.pin
    Contoh : GPIN.SaMurai7094.1234
    Catatan : PIN Bebas boleh angkan / huruf disarankan kominasi angkahuruf dan panjang agar aman

  • Tambah Hp
    Format : THP.nomorhpbaru.pin
    Contoh : THP.0877412345.1234

  • Daftar Downline
    Format : REG.nama.alamat.nohpdownline.markup.pin
    Contoh : REG.Kios Cell.Bandung.0812222122.50.1234

  • Kirim Saldo
    Format : TS.id_member.jumlah.pin
    Contoh : TS.OK001.200000.5555

  • Cek Status Transaksi
    Format : CEK.TUJUAN
    Contoh : CEK.0812345789

  • Komplain
    Format : CS.isipesan.pin
    Contoh : CS.isi 10rb ke 0812222121 blm mask tgl.18 tolong bntu.1234

  • Tiket Deposit
    Format : TIKET.Jumlah.pin
    Contoh : TIKET.500000.1234

  • BlokirDownline
    Format : OFF.id_agen.pin
    Contoh : OFF.BJ003.pin

  • Buka Blokir
    Format :ON.id_agen.pin
    Contoh :ON.BJ003.1234

  • UbahUp/Mark up Harga
    Format : UBAHMARKUP.id_downline.markup.pin
    Contoh : UBAHMARKUP.IN002.100.1234

  • UbahUp/Mark up Perkodeproduk
    Format : UMP.id_agen.kode-prodak.markup.pin
    Contoh : UMP.TE001.S5.50.1234

  • Cek Tagihan PPOB
    Format : CEK.PLN/ TELKOM/ BPJS/ HALO/ PDAM/ SPEEDY.id_pelanggan.pin
    Contoh : CEK.PLN.53531004567699.1234

  • Bayar Tagihan PPOB
    Format : BAYAR.id_pelanggan.pin
    Contoh : BAYAR.535210045676.1234

  • Hapus Nomor Paralel
    Format : HAPUSNOHP.nomorparalel.pin
    Contoh : HAPUSNOHP.0877788882222.pin

  • Cek Semua Nomor
    Format : LP.pin
    Contoh : LP.1234

  • Ganti Nama Member
    Format : GN.namabaru.pin
    Contoh : GN.AbadiCell.1234

  • Ambil Bonus REBATE
    Format : BONUS.pin
    Contoh : BONUS.1234

  • ADD JABBER
    Format : TIM.id/account.pin
    Contoh : TIM.joko@jabbim.com/Gf^%GYFRUYFyf.pin

  • ADD TELEGRAMBOT
    Format : TL.ID telegram.pin
    Contoh : TL.212121212.pin

  • ADD WhatsApp / WA
    Format : WA.NOMER WA.pin
    Contoh : WA.6281111111112.pin
